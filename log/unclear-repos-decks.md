# Unclear repos: what they are

## distributors

**Distributors** ⟜ Haskell library for profunctors and invertible parsers/printers

**Math-inspired** ⟜ Unifies parsing, printing, and grammar via distributor theory (profunctors on distributive categories)

**Published & maintained** ⟜ On Hackage, open source, active GitHub repo with CI

**Optics & invertibility** ⟜ Implements grates, monocles, withers; bidirectional programming patterns

**Worth keeping**: Yes. This is a real, published Haskell library. Should be in ~/haskell/.

## skills

**Skills** ⟜ Anthropic's official repository of Claude agent skills (example implementations)

**Reference material** ⟜ Demonstrates skill patterns for creative, technical, enterprise, and document tasks

**Educational** ⟜ Published as-is for learning; implementations may differ from production Claude

**Not yours** ⟜ External repo (Anthropic), included for reference and plugin marketplace access

**Worth keeping**: Yes, but belongs in ~/other/ (external reference material, not your personal work).

**Multi-platform** ⟜ Claude Code plugin, Codex, OpenCode versions
