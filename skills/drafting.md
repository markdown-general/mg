drafting ⟜ a common pattern for agents & operatots drafting a card
⟜ read into a set of generalist cards that the draft will touch.
⟜ read external core draft material and converse.
⟜ establish a baseline for where the draft meshes with other cards.
⟜ write a deck establishing the structure of the draft
⟜ check this deck for coherence; semantic coverage and drift.
⟜ repeat at a lower level ι write a deck elaborating a line in the main deck.
⟜ think of this entire draft as a tree of elaborations, in states of ι verification | semantic analysis | testing | syntactic analysis
