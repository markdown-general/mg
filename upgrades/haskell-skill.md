Haskell libraries tend to go in ~/repos/. The repos there are mostly Haskell libraries and can be a good resource for pattern matching solutions.

Default choices:

⟜ language: GHC2024
⟜ cabal stanzas for options and extensions from ~/repos/numhask-space
⟜ CI as per numhask-space
⟜ doctests in haddocks and no test stanza. testing happens using an independent test runner: cabal-docspec
